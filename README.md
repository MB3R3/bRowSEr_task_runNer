# Browser Task Runner

A modular browser automation framework built with Python and Playwright.

## Features (Planned)

- Login Automation
- Screenshot Capture
- Web Scraping
- File Downloads
- HTML Reports
- Logging
- Batch Execution
- Scheduler
- SQLite History
- FastAPI Integration
- Streamlit Dashboard

## Project Structure

```text
browser-task-runner/
```

## Setup

```bash
python -m venv venv

pip install -r requirements.txt

playwright install
```
